Components

Kune UI leverages JSX to provide common-use components with useful out-of-the-box functionalities.

As such, you might notice that most Kune UI components replace common HTML tags. You can still use every-day HTML markup along with Kune UI, but we strongly encourage you to check out the simple yet powerful functionalities we have implemented so that you can focus on building cool and useful apps.

Typography

A UI kit is only as strong as its ability to render content in a satisfactory manner. Kune UI is no exception to this rule.

Here's a list of the typography components the Kune UI team has implemented along with the common HTML tags they're meant to replace:

  • Heading (h1, h2, h3, h4, h5, h6)
  • Paragraph (p)
  • Separator (hr)
  • Link (a)
  • Accent (b, strong, i, em, mark, small, del, ins, sub, sup)
  • List (ul, ol, dl)
  • Citation (abbr, address, bdo, blockquote, cite, q)
  • Image (img)

Heading

The <Heading/> component provides a very handy feature out of the box: automatic named anchors.

Named anchors are useful to be able to link directly to content using url fragment identifiers (the hashtag portion of a link, i.e. "#my-title").

For example, let's say that you own the site example.com and in it you place the following code:

<Heading>My Title</Heading>

The previous JSX syntax generates the following markup:

<h1 class="heading-1">
  <a name="my-title"></a>
  My Title
</h1>

This means that you can link to that header directly from the url like this: https://example.com#my-title

You may wonder what happens with headings that contain special characters. Kune UI's <Heading/> component takes care of normalizing special characters so that the generated named anchor conforms to URL standards.

Let's say that you have a title with the hungarian phrase Árvíztűrő tükörfúrógép!, as such:

<Heading>Árvíztűrő tükörfúrógép!</Heading>

The previous JSX syntax generates the following markup:

<h1 class="heading-1">
  <a name="arvizturo-tukorfurogep"></a>
  Árvíztűrő tükörfúrógép!
</h1>

This is not only highly convenient, but extremely valuable for SEO purposes.
